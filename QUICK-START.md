## 🛠️ **Troubleshooting**

### **Common Issues & Solutions**

**❌ HTML Triggers Not Working**
- Verify `.cursor/rules/cursor-rules-automation.mdc` is installed
- Check file contains HTML comment triggers: `<!-- AI CUSTOMIZATION TRIGGER: -->`
- Restart Cursor to reload rules

**❌ Meta-Intelligence Not Responding**
- Ensure `.cursor/rules/cursor-rules-advanced-automation.mdc` is active
- Test with specific question: *"What implementation gaps do you detect in this system?"*
- Check Cursor rules are loading properly

**❌ Performance Not Improving**
- Verify `.cursorindexingignore` is in project root
- Check token count in Cursor status bar
- Restart Cursor to apply indexing changes

**❌ Templates Seem Empty**
- Templates are designed with HTML triggers for guided setup
- Ask AI: *"Please help me customize this template"*
- Look for `<!-- AI CUSTOMIZATION TRIGGER: -->` comments
